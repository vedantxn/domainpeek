domainpeek

Keyless domain intelligence from the command line and for AI agents. No API keys, no config.

Quick start

npx domainpeek check stripe.com

Availability, price, register link, and RDAP/DNS intel for one domain.

npx domainpeek find lumind

Check a name across every priced TLD; cheapest available flagged, plus npm/GitHub brand check.

npx domainpeek guard stripe

See which typo and other-TLD variants are already registered.

Commands

check — availability, registrar pricing, checkout_url, and registration/DNS intel.

domainpeek check startup.ai
domainpeek check startup.ai,startup.io,startup.dev   # batch, max 10
domainpeek check startup.ai --table                  # human-readable
domainpeek check startup.ai --fast                   # availability only
domainpeek check startup.ai --deep                   # + TLS, Wayback, crt.sh

find — one bare name across priced TLDs.

domainpeek find startup
domainpeek find startup --tlds com,io,dev,ai

guard — defensive scan for typo and other-TLD variants.

domainpeek guard startup
domainpeek guard startup --tlds com,io,dev

Output

  • JSON by default when output is piped
  • --table for human-readable output in a terminal
  • Exit codes: 0 all resolved, 1 partial failure, 2 total failure
